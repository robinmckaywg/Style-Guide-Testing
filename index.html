Bootstrap 4 Style guide

Style guide powered with nunjucks and Sass.

Read more about Bootstrap 4 at http://getbootstrap.com/. Bootstrap 4 is a major rewrite of almost the entire project.

Bootstrap 4 also drops the Affix plugin, this template shows how to use `position: sticky;` as a replacement for it. You should use the sticky polyfill for full browser support.
